# 🔢 Modern Calculator App

A sleek, fully responsive calculator web application with dark/light theme, memory functions, and full keyboard support.


## ✨ Features

- ➕ Basic arithmetic operations (+, -, ×, ÷)
- 🧠 Memory storage (M+, M-, MR) – accessible via keyboard
- 🌗 Dark / Light theme toggle
- ⌨️ Full keyboard support (numbers, operations, delete, escape)
- 📱 Responsive design – works on mobile, tablet, and desktop
- 🎯 Error handling (division by zero, invalid operations)
- 🔘 Ripple effect on button clicks
- 💾 Memory indicator showing stored value

## 🖥️ Live Demo

Open `index.html` in any modern browser – no build steps or dependencies required.

## 🚀 How to Use

### On‑screen Buttons

| Button | Action |
|--------|--------|
| `AC`   | Clear everything (reset calculator) |
| `DEL`  | Delete the last digit |
| `÷` `×` `-` `+` | Choose an operation |
| `=`    | Calculate the result |
| `0`–`9` `.` | Enter numbers |

### ⌨️ Keyboard Shortcuts

| Key(s) | Action |
|--------|--------|
| `0`–`9` `.` | Type numbers |
| `+` `-` `*` `/` | Operations (`*` = ×, `/` = ÷) |
| `Enter` or `=` | Compute result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all (AC) |

### 🧠 Memory Functions (Keyboard Only)

| Shortcut | Action |
|----------|--------|
| `Ctrl` + `M` | Add current display value to memory (M+) |
| `Ctrl` + `Shift` + `M` | Subtract current display value from memory (M−) |
| `Alt` + `M` | Recall memory value (MR) |

The memory indicator at the bottom shows the current stored value or "No Memory" when empty.

### 🌗 Theme Toggle

Click the **moon/sun icon** in the top‑right corner to switch between light and dark mode. The theme preference is not saved after page refresh.

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling, grid layout, animations, dark mode
- **JavaScript (ES6)** – Calculator logic, event handling, keyboard support
- **Font Awesome 6** – Icons for memory and theme toggle
